📺 Hack Stream — документация проекта
📌 Описание
Hack Stream — это интерактивная веб-игра для стримов (OBS + YouTube/Twitch). Игроки пишут коды в чат или на сайте, пытаясь «взломать систему». Каждый раунд длится ограниченное время, после чего генерируется новый код.
🎮 Основная идея
Есть 4-значный секретный код
Игроки пытаются его угадать
Каждый раунд даются подсказки
За правильный ответ игрок получает очки
Есть таблица лидеров
⏱ Механика раунда
Генерируется новый код (1000–9999)
Выбираются случайные 3–4 подсказки
Запускается таймер (обычно 20 секунд)
Игроки отправляют попытки
Проверяется правильность
Запускается новый раунд
💡 Подсказки (примеры)
Первая цифра
Последняя цифра
Сумма цифр
Чётные/нечётные числа
Количество цифр
